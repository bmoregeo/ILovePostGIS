I cannot talk about PostGIS without first going through PostgreSQL. Postgre is a database management system that competes in the same space as MySQL. Postgre is cooked in with Indexing, many different data types incuding JSON, and a programming language PL/pgSQL that is similar to TSQL. I became interested in PostgreSQL back in 2009 when Oracle bought out MySQL. I was a huge MySQL fan, but I really didn't want to deal with Oracle. Subsequently a lot of people felt like me and MariaDB was spun off of MySQL. PostgreSQL, through PostGIS, also has SUBSTANTIALLY better spatial capabilities compared to MySQL. You can pretty much do anything in PostgreSQL with regards to spatial analysis. Back at the time I forked from MySQL, it couldn't even do a proper point-in-polygon.

postgres on github

The best part of Postgre is that it is open source. This has allowed a bunch of crazy projects to prosper, like PostGIS for spatial databases, PGRouting for vehicle routing, HStore for schema-less data storage, and many more that I don't use.

Postgres is awesome, but that is not why I'm here... PostGIS is a spatial extension to Postgre. At the lowest level Postgre allows us to define spatial data types, point; line; polygon, build spatial indexes, and manage projections.

BUT WAIT THERE'S MORE!!!
Personally, the hardest part of performing analysis is keeping to a single set of directions all the way through. I ALWAYS end up forgetting a step and waste a half a days work. This is partially what got me into learning python. THEN I realized I could do 90% of the stuff I was doing in python with a couple SQL queries. 
SELECT room.*
FROM 
	(SELECT shape, objectid	FROM room WHERE OBJECTID = 2459) selected, 
	room
WHERE
	ST_3DDWithin(room.shape, selected.shape, 10)
	AND NOT selected.objectid = room.objectid;
The best part about this is that I don't create a bunch of these little shapefiles called, "Buffer 1.shp", "Buffer 2.shp". I was also traveling while I made this slide and didn't have 3D Analyst extension checked out so I couldn't test this model out!
